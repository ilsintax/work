급여대장 업로드 편집기

급여 데이터를 붙여넣거나 불러온 뒤 제목을 정리하고, 합계와 숫자 서식이 적용된 깔끔한 엑셀 파일로 내보낼 수 있습니다.

Input
Excel / Paste
Transform
Header / Salary
Export
Styled Excel

1단계: 데이터 입력

📁 방법 A. 엑셀 파일 업로드

📋 방법 B. 복사 & 붙여넣기 (추천)

2단계: 제목(헤더) 행 선택

제목에 해당하는 행을 모두 클릭하세요. 병합된 제목도 똑똑하게 한 줄로 압축해 줍니다.
※ 헤더 선택시 제목은 제일 밑에 내용이 우선순위로 표시됩니다.

3단계: 빈 행 삭제를 위한 '기준 열' 선택

이름, 사번 등 데이터가 없으면 무의미한 열을 1개 클릭하세요. 빈 행이 모두 날아갑니다.

4단계: 1차 사원 기본정보 열 확정

클릭은 한 열 선택, Shift+클릭은 구간 선택, Windows는 Ctrl+클릭, Mac은 Cmd+클릭으로 여러 열을 선택하거나 해제할 수 있습니다. 여기서 선택한 기본정보 열은 다음 단계 아래 칸에 합쳐집니다.

5단계: 2차 급여정보 열 합치기 및 0 치환

클릭은 한 열 선택, Shift+클릭은 구간 선택, Windows는 Ctrl+클릭, Mac은 Cmd+클릭으로 여러 열을 선택하거나 해제할 수 있습니다. 완료를 누르면 현재 보이는 값 없는 칸과 문자는 0으로 바뀝니다.
