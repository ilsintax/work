# Auto Classifier

Static HTML app for Excel-based classification and summary work.

## Files

- `index.html` : main app
- `xlsx.full.min.js` : local Excel library file
- `THIRD_PARTY_NOTICES.md` : third-party license notes

## Deploy

If your repository is `work`, publish the repository root with GitHub Pages and open:

- `https://<your-account>.github.io/work/auto-classifier/`

## Rule Set Files

- You can rename exported rule set JSON files freely.
- Import works by reading the JSON content, not the filename.

## Current dependency status

- Local: `xlsx.full.min.js`
- CDN: Tailwind CSS, Tabulator, Google Fonts
