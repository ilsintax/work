Simple Workflow v015

자동분류 및 요약 도구

엑셀을 올리거나 붙여넣고, 제목행과 필수열, 분류기준 열을 클릭으로 확정한 뒤 규칙을 만들고 미분류만 다시 검토하는 흐름으로 정리했습니다.

입력
업로드 + Ctrl+V
설정
제목행 / 필수열 / 기준열
분류
규칙 + 추천
엑셀 업로드 또는 Ctrl+V 붙여넣기로 작업을 시작해 주세요.

엑셀 불러오기

업로드 또는 붙여넣기 중 편한 방법으로 데이터를 넣어 주세요.

불러온 파일
선택된 파일이 없습니다.
붙여넣기 영역
엑셀 범위를 복사한 뒤 아래를 클릭하고 Ctrl+V를 눌러 주세요.
제목행 후보
미리보기를 보고 제목행을 직접 선택해 주세요.
분류기준 열
적요, 내용, 거래처처럼 분류에 쓰일 열입니다.
필수열
값이 비면 해당 행을 무시할 기준 열입니다.
열 선택
열 이름을 눌러 분류기준에 넣고, 필요하면 필수열로도 지정해 주세요.
미리보기
위에서 제목행과 열을 정한 뒤, 아래 넓은 작업 영역에서 현재 구성을 바로 확인합니다.
데이터를 불러오면 여기에 미리보기가 표시됩니다.

분류과목 관리

필요한 분류과목을 먼저 추가해 두면 빠른 규칙 저장과 결과 화면에서 바로 선택할 수 있습니다.

저장된 규칙 목록

확인으로 저장한 규칙만 여기 모입니다. 목록을 누르면 상세 내용을 보고, 필요하면 불러와 수정할 수 있습니다.

저장된 규칙을 선택하면 여기에서 상세 내용을 확인합니다.

빠른 규칙 만들기

표에서 셀을 클릭하면 조건이 쌓입니다. 분류과목을 고르고 확인을 누르면 저장된 규칙 목록으로 이동합니다.

현재 조건
규칙 작업 미리보기
빠른 규칙을 먼저 만들고, 아래 넓은 미리보기에서 미분류 행을 검색하고 정렬하면서 셀 클릭으로 조건을 추가할 수 있습니다.
헤더 클릭: 정렬 헤더 검색칸: 즉시 필터 셀 클릭: 빠른 규칙 조건 추가

분류 결과

저장된 규칙과 추천을 바탕으로 분류하고, 미분류만 다시 검토합니다.

검색 / 필터 / 일괄 채우기
검색과 정렬을 하면서 현재 보이는 행을 한 번에 선택하고 분류과목을 채울 수 있습니다.
열별 상세 필터는 그리드 헤더의 검색칸을 사용해 주세요.
체크한 행 또는 현재 보이는 전체 선택을 기준으로 먼저 임시 채우기를 한 뒤, 확인을 눌러 확정할 수 있습니다.
헤더 클릭: 정렬 헤더 검색칸: 즉시 필터 체크 후 Ctrl + D: 선택행 채우기 셀 클릭: 빠른 규칙 조건 추가

요약

분류과목별 건수와 금액 합계를 확인합니다.

분류 후 요약이 표시됩니다.

저장

현재 제목 설정, 규칙, 분류과목을 저장하거나 다시 불러옵니다.

저장 요약

현재 작업 상태를 한눈에 확인할 수 있습니다.
