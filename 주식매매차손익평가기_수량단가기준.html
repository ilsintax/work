Stock Worksheet Builder

주식 매매차손익 평가기

엑셀 파일을 업로드한 뒤 주식명 셀, 제목행, 거래일자, 적요, 차변, 대변, 잔액을 선택하면 각 시트의 주식명을 읽어 종목별 상세 시트와 일자별 손익집계 시트를 새 엑셀로 만듭니다.

처리 기준

차변과 대변은 방향 판단용이며, 거래금액은 적요의 수량 × @단가로 계산합니다.

Tutorial

처음 사용 전 꼭 확인할 내용

아래 규칙으로 적요를 읽고, 위하고 또는 세무사랑에서 내려받은 엑셀을 기준으로 처리합니다.

대표 인식 예시

테라뷰 / 1288주 / @10120

적요 인식 규칙

oo주 / @단가 순서인 행만 인식

주식 매수 및 매도 시 적요 안에 oo주 / @단가가 반드시 이 순서로 있어야 합니다.

주식수주 다음에 @단가가 이어지지 않으면 해당 행은 계산에서 제외합니다.

적요를 인식하면 그 행의 매수 및 매도가액은 주식수 × @단가로 계산합니다.

(매매차익), (수수료), (제세금)으로 시작하는 적요는 재업로드 시 중복 계산되지 않도록 무시합니다.

미리보기 기준

처음 지정할 때는 첫 시트만 대표로 표시

거래처 및 일자 등의 형식이 모든 시트에서 동일하다고 보고, 선택 단계에서는 첫 시트만 보여줍니다.

화면에는 첫 시트만 보이지만, 선택한 좌표는 전체 시트에 동일하게 적용됩니다.

위하고 사용 시 엑셀 다운로드 방법

위하고

  1. 1. 거래처원장으로 이동합니다.
  2. 2. 주식 계정과목 입력 후 내용을 선택합니다.
  3. 3. 인쇄를 누릅니다.
  4. 4. 전체내용순백지 출력을 선택합니다.
  5. 5. 엑셀버튼을 누릅니다.
  6. 6. 그룹별 시트 분할 인쇄로 저장합니다.

세무사랑 사용 시 엑셀 다운로드 방법

세무사랑

  1. 1. 거래처원장으로 이동합니다.
  2. 2. 주식 계정과목 입력 후 내용을 선택합니다.
  3. 3. 거래처 처음부터 끝까지를 선택합니다.
  4. 4. 상단의 CF9 일괄출력을 누릅니다.
  5. 5. 엑셀저장으로 내려받습니다.

사용 파일 기준

위의 방식으로 저장한 엑셀을 이 프로그램에 업로드해서 사용합니다.

1. 파일 업로드

XLSX/XLS 파일을 읽고, 형식 지정 단계에서는 첫 시트만 대표 미리보기로 보여줍니다.

2. 기준 위치 선택

대표로 보이는 첫 시트에서 기준 좌표를 선택하면 같은 좌표를 다른 시트에도 적용합니다.

최종잔액 0인셀

선택 중에는 미리보기에서 셀을 클릭하세요. 제목행은 행 아무 셀이나 눌러도 됩니다.

3. 종목별 기초값 입력

시트별로 읽은 주식명을 확인하고 기초주식수와 기초취득원가를 입력하세요.

원본 시트명 주식명 인식 거래행 기초주식수 기초취득원가 메모

안내

기초주식수가 0보다 큰 종목은 정확한 매도손익 계산을 위해 기초취득원가도 함께 입력하는 것이 안전합니다.

4. 결과 다운로드

종목별 상세 시트와 일자별 손익집계 시트가 포함된 엑셀입니다.

엑셀 다운로드

5. 추가작업

기존 손익 계산 결과는 그대로 두고, 회계프로그램 업로드용 파일을 추가로 만들 수 있습니다.

생성 가능

손익계산 / 매매차익 업로드 / 평가차익 업로드

추가작업 여부(매매차익 회계프로그램 업로드)

예를 선택하면 공통 계정과목과 업로드용 파일 생성 옵션이 열립니다.

공통 필수 입력

회계 업로드 파일에서 사용할 계정과목 코드와 계정과목명을 먼저 입력합니다.

업로드 파일 생성

버튼을 눌러 업로드 형식별 추가 입력을 진행합니다.

세무사랑 필수 입력

회사명과 사업자등록번호는 헤더 셀에, 해당연도는 매매일자의 월/일과 합쳐 yyyymmdd를 만드는 데 사용합니다.

세무사랑 업로드 엑셀 다운로드

위하고 생성 안내

위하고는 회사명, 사업자등록번호, 연도 입력 없이 생성합니다. 날짜는 매매일자의 월과 일을 분리해서 A열 월 / B열 일에 넣습니다.

입력 열

A 월, B 일, C 구분, D 계정과목코드, E 계정과목명, F 거래처코드, G 거래처명, H 적요명, I 차변, J 대변

위하고 업로드 엑셀 다운로드

6. 평가차익 추가작업

평가차액 입력을 완료한 경우에만 회계프로그램 업로드 파일을 추가로 만들 수 있습니다.

반영 기준

종목별 요약의 평가차액을 기준으로 업로드 행을 만듭니다.

추가작업 여부(평가차익 회계프로그램 업로드)

`예`를 선택하면 평가연월일과 평가 계정과목을 입력한 뒤 업로드 파일 생성 버튼을 열 수 있습니다.

공통 필수 입력

주식 계정과목, 평가연월일, 평가이익/손실 계정을 먼저 입력한 뒤 확인 버튼을 누르세요.

업로드 파일 생성

확인 후 형식별 생성 버튼을 눌러 평가차익 업로드 파일을 만듭니다.

세무사랑 필수 입력

회사명과 사업자등록번호를 입력하면 평가연월일 기준 세무사랑 업로드 파일을 생성합니다.

평가차익 세무사랑 업로드 엑셀 다운로드

위하고 생성 안내

평가연월일의 월/일을 사용해 평가차익 위하고 업로드 파일을 생성합니다.

입력 열

A 월, B 일, C 구분, D 계정과목코드, E 계정과목명, F 거래처코드, G 거래처명, H 적요명, I 차변, J 대변

평가차익 위하고 업로드 엑셀 다운로드
