# 손익보고서 생성기

GitHub Pages에 바로 올릴 수 있도록 정리한 정적 HTML 버전입니다.

## 포함 파일

- `index.html`
- `report-builder.js`
- `.nojekyll`

## 사용 방법

1. GitHub 저장소를 생성합니다.
2. 이 폴더의 파일을 저장소 루트 또는 `docs` 폴더에 업로드합니다.
3. GitHub 저장소의 `Settings > Pages`에서 배포 경로를 선택합니다.
4. 배포가 끝나면 생성된 URL로 접속합니다.

## 화면 흐름

1. 엑셀 파일 업로드
2. `손익보고서 생성` 버튼 클릭
3. `데스크탑용 HTML 다운로드` 또는 `모바일용 HTML 다운로드` 클릭

## 참고 사항

- 업로드한 엑셀 파일은 브라우저 안에서만 처리됩니다.
- 외부 서버로 엑셀 파일을 전송하지 않습니다.
- 엑셀 파싱은 SheetJS CDN을 사용하므로 인터넷 연결이 필요합니다.
- 다운로드된 HTML은 거래처 메일 또는 카카오톡 전달용으로 사용할 수 있습니다.
