1. 엑셀 데이터 붙여넣기

엑셀에서 변환할 표(이름, 금액 등)를 복사(Ctrl+C)하여 아래 공간에 붙여넣기(Ctrl+V) 하세요.

2. 제목 행(Header) 선택

데이터의 기준이 되는 제목 행을 클릭해 주세요. 클릭한 행을 기준으로 표가 정리됩니다.

3. 이름 / 금액 열 지정

이름 열과 금액(세후) 열의 제목을 순서대로 클릭하세요.

금액: 미선택

4. 추가 데이터 열 선택 (선택사항)

결과 표에 함께 표시하고 싶은 추가 열(부서, 직급 등)이 있다면 제목을 클릭하세요.
